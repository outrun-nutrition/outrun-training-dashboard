# Outrun Training Dashboard

Personal training dashboard — HYROX + Marathon prep 2026.

**Live**: https://outrun-nutrition.github.io/outrun-training-dashboard/

## What this is

Private-use training tracker with:
- 35-week periodized plan
- Daily workouts + templates
- RPE / ankle / HRV / sleep logging
- Google Sheets cloud sync (via Apps Script)
- Apple Health auto-sync (iOS Shortcut)

## Usage

Open the Live URL, go to 參考 tab → ☁️ 雲端同步 → 設定 URL with your Apps Script Web App endpoint.

## Source

Canonical source lives in a private workspace. This repo is the public deploy target for GitHub Pages.
